# SpendWise

SpendWise is a modern Android expense tracker designed to help users manage daily spending, categorize transactions, and monitor total expenses with a clean, dark-themed interface.

## Features

- Record expense title, amount, category, and notes
- Display recent transactions with totals and counts
- Delete expenses directly from the transaction list
- Persistent storage with Room database
- Material 3-inspired dark UI

## Project structure

- `app/`: Android application module
- `app/src/main/java`: Kotlin source code
- `app/src/main/res`: UI layouts, values, and resources

## Build

1. Install Android Studio with the Android SDK
2. Open the project root in Android Studio
3. Build and run the `app` module

# SpendWise

SpendWise is a modern Android expense tracker app built with Kotlin and Android Jetpack. It helps users track daily spending, manage a monthly salary budget, and review spending analytics with charts and smart insights.

## Features

- Add and categorize expenses by day
- Track daily spending versus a dynamic budget
- View spending score, mood tag, and budget progress
- See category breakdown with donut charts
- Review weekly spending trends and monthly health grade
- Store expenses locally using Room database

## Project structure

- `app/` - Android application module
- `app/src/main/java` - Kotlin source code, ViewModel, database, repository, custom UI views
- `app/src/main/res` - layouts, drawables, colors, and resources
- `README.md` - project overview and build instructions
- `.gitignore` - ignores build artifacts and local files

## Build instructions

1. Open the project in Android Studio.
2. Sync Gradle.
3. Run the app on an emulator or connected device.

## Notes

- This repo is public and ready to use as an Android expense tracker.
- Use the bottom floating button to add expenses and switch tabs to view analytics.
